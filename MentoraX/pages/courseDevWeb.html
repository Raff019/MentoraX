Bem-vindo ao Curso de Desenvolvimento Web!

Este curso abrangente irá guiá-lo pelos fundamentos e pelas tecnologias essenciais para construir aplicações web modernas e interativas. Você aprenderá desde a estrutura básica de uma página até a criação de interfaces dinâmicas e a interação com servidores.

O Que Você Vai Aprender:

Módulo 1: Fundamentos da Web

  • Como a internet funciona: protocolos HTTP, DNS e navegadores
  • Introdução ao HTML: estrutura básica de documentos web, tags e atributos
  • Formulários HTML: coleta de dados do usuário
  • Semântica HTML5: tags para estruturar o conteúdo de forma significativa
  • Acessibilidade web (WCAG): princípios básicos

Módulo 2: Estilização com CSS

  • Introdução ao CSS: seletores, propriedades e valores
  • Modelo de caixa (Box Model): padding, margin, border
  • Layouts com CSS:
  • Fluxo normal e posicionamento (static, relative, absolute, fixed)
  • Flexbox: criando layouts flexíveis e responsivos
  • Grid CSS: layouts complexos em duas dimensões
  • Responsividade: Media Queries e adaptação a diferentes dispositivos
  • Animações e transições CSS para interfaces dinâmicas

Módulo 3: Interatividade com JavaScript

  • Introdução ao JavaScript: sintaxe básica, variáveis, tipos de dados
  • Estruturas de controle (condicionais e loops) em JavaScript
  • Manipulação do DOM (Document Object Model): modificando elementos HTML e CSS com JS
  • Eventos: como reagir às ações do usuário (clicks, mouseovers, etc.)
  • Funções em JavaScript: reutilização de código
  • Introdução a APIs web (ex: Fetch API para requisições HTTP)

Módulo 4: Desenvolvimento Front-End Avançado (Introdução)

  • Introdução a frameworks e bibliotecas front-end (visão geral de React, Angular ou Vue.js)
  • Gerenciamento de estado (conceitos básicos)
  • Componentização de interfaces
  • Ferramentas de build (ex: npm, yarn)
  • Noções de testes front-end

Módulo 5: Desenvolvimento Back-End (Introdução)

  • Introdução ao desenvolvimento back-end: o que é e para que serve
  • Visão geral de linguagens e frameworks back-end (ex: Node.js, Python/Django/Flask, PHP/Laravel)
  • Bancos de dados: conceitos básicos e tipos (SQL e NoSQL)
  • Comunicação front-end e back-end: APIs REST
  • Autenticação e autorização (conceitos básicos)

Módulo 6: Ferramentas e Fluxo de Trabalho

  • Sistemas de controle de versão com Git e GitHub
  • Editores de código e IDEs para desenvolvimento web
  • DevTools do navegador para depuração e análise
  • Hospedagem e deploy de aplicações web

Ao concluir este curso, você terá uma base sólida para construir suas próprias aplicações web, entender o fluxo de desenvolvimento e continuar explorando as diversas áreas do desenvolvimento web. Prepare-se para criar experiências digitais incríveis!
